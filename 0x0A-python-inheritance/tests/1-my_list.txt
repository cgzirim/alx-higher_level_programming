======================
How to use 1-my_lis.py
======================

This module defines a class ``MyList`` that inhereits from ``list``.

Usage:
======

	>>> MyList = __import__('1-my_list').MyList

	>>> my_list = MyList()

	>>> my_list
	[]

	>>> my_list.append(1)
	>>> my_list.append(4)
	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(5)
	
	>>> my_list
	[1, 4, 2, 3, 5]

	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]

	>>> my_list
	[1, 4, 2, 3, 5]

	>>> my_list = MyList([3, 4, 2])

	>>> my_list
	[3, 4, 2]

	>>> my_list.print_sorted()
	[2, 3, 4]

	>>> my_list = MyList("happy")

	>>> my_list
	['h', 'a', 'p', 'p', 'y']

	>>> my_list.print_sorted()
	['a', 'h', 'p', 'p', 'y']

	>>> new_list = MyList()

	>>> new_list
	[]
